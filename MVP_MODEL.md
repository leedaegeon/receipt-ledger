# MVP 모델

## 1) 문제/가설
- 문제: 가계부 포기 이유는 "입력이 귀찮음"
- 가설: 영수증 자동입력으로 입력 시간을 10초 이내로 줄이면 주간 유지율이 증가한다.

## 2) 핵심 기능 (MVP)
1. 영수증 촬영/업로드
2. OCR 텍스트 추출
3. 결제일/금액/점포명 자동 파싱
4. 카테고리 자동 태깅
5. 검수/수정 후 저장
6. 월별 합계/카테고리 통계

## 3) 제외 기능 (후순위)
- 계좌/카드 연동
- 클라우드 동기화
- 가족 공유
- AI 소비 코칭

## 4) 사용자 플로우
1. 홈에서 [영수증 추가]
2. 촬영 또는 갤러리 선택
3. 자동 추출 결과 확인
4. 틀린 값 수정
5. 저장 → 목록/통계 반영

## 5) 성공 지표 (초기 4주)
- D1 리텐션: 35%+
- 영수증 저장 성공률: 90%+
- 자동 추출 후 수정 없는 비율: 60%+
- 평점: 4.3+

## 6) 수익 모델
- 유료 단품: 2,900원 (추천)
- 대안: 무료 라이트 + Pro 업그레이드
