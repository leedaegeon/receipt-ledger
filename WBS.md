# WBS (2주 MVP) — 월간 계좌내역 자동 가계부

## Week 1: 파싱/정규화 코어
- D1: 요구사항 고정, 샘플 파일 수집(은행/카드 각 5종)
- D2: 프로젝트 세팅 (Kotlin + Compose + Room)
- D3: 파일 업로드 UI (PDF/XLSX/CSV)
- D4: 파서 v1
  - PDF 텍스트 추출
  - XLSX/CSV 리더
- D5: 컬럼 자동 매핑 + 수동 매핑 UI
- D6: 거래 정규화 엔진 (날짜/금액/적요 표준화)
- D7: 중복 제거 로직 + 오류 행 리포트

## Week 2: 분류/리포트/출시 준비
- D8: 카테고리 룰 엔진 v1 (키워드/정규식)
- D9: 분류 검수/수정 화면 + 수정 학습 저장
- D10: 월간 리포트 생성
  - 총수입/총지출/순현금흐름
  - 카테고리 비중
  - 상위 지출 Top N
- D11: 고정비 후보 탐지(반복 거래 추정)
- D12: 성능/예외 처리 (대용량 5,000행)
- D13: QA/테스트(샘플 파일 10종), 버그 수정
- D14: 릴리즈 빌드 + 스토어 리스팅 초안

## 우선순위
- P0: 업로드 → 파싱 → 정규화 → 분류 → 리포트 end-to-end
- P1: 수동 매핑 UX, 오류 행 처리, 성능 안정화
- P2: UI 폴리싱, 내보내기 개선, ASO 문구

## 예상 소요시간
- 개발: 55~70시간
- QA/데이터 검증: 12~16시간
- 배포 준비: 6~8시간
