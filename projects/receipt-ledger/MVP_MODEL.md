# MVP 모델

## 1) 문제/가설
- 문제: 사용자는 가계부를 꾸준히 쓰기 어렵고, 월말 정산에 시간이 많이 든다.
- 가설: 한 달치 계좌내역(PDF/엑셀/CSV) 업로드만으로 자동 분류/분석을 제공하면, 월간 정산 시간을 70% 이상 줄일 수 있다.

## 2) 핵심 가치 제안
- 입력 최소화: 수기 입력 없이 파일 업로드 중심
- 자동화: 파싱 → 분류 → 월간 리포트 생성
- 통찰 제공: 카테고리 비중, 고정비 후보, 상위 지출 자동 표시

## 3) 핵심 기능 (MVP)
1. 파일 업로드: PDF/XLSX/CSV
2. 컬럼 자동 인식 + 수동 매핑 UI
3. 거래 정규화(날짜/금액/적요 표준화)
4. 중복 제거(Dedup)
5. 카테고리 자동 분류(룰 기반 + 사용자 수정 학습)
6. 월간 리포트 생성
   - 총수입/총지출/순현금흐름
   - 카테고리별 지출 비중
   - 고정비 후보
7. 분류 검수/수정 화면

## 4) 제외 기능 (후순위)
- 은행/카드 실시간 연동(Open Banking)
- 클라우드 동기화/멀티디바이스
- 가족 공유
- 고급 AI 코칭(예산 자동 추천, 이상치 탐지)

## 5) 사용자 플로우
1. [파일 가져오기] 클릭
2. 파일 선택(PDF/XLSX/CSV)
3. 자동 파싱 결과 확인
4. 컬럼 매핑/오류 행 확인
5. 자동 분류 결과 검수(필요 시 수정)
6. [월간 리포트 생성]
7. 요약/카테고리/상위지출 확인 및 내보내기

## 6) 성공 지표 (초기 4주)
- 파일 파싱 성공률: 90%+
- 자동 분류 정확도(수정 전 기준): 75%+
- 리포트 생성 완료율: 95%+
- 사용자 월간 정산 시간 절감 체감: 70%+
- 앱 평점: 4.3+

## 7) 수익 모델
- 유료 단품(권장): 4,900원
  - 장점: 서버비 없음, 유지보수 단순
- 대안: 무료 체험(월 1회) + Pro 영구 구매

## 8) 출시 기준 (Go/No-Go)
- 주요 샘플 파일 10종 이상 정상 처리
- 크래시 프리 세션 99%+
- 치명적 데이터 오류(금액/날짜 전환 오류) 0건
